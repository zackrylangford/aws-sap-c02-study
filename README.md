# AWS Solutions Architect Professional (SAP-C02) Study Guide

Self-study materials and notes for the AWS Certified Solutions Architect - Professional exam.

## Exam Overview

- **Exam Code**: SAP-C02
- **Duration**: 180 minutes
- **Format**: Multiple choice and multiple response
- **Passing Score**: 750/1000

## Exam Domains

1. **Design Solutions for Organizational Complexity** (26%)
2. **Design for New Solutions** (29%)
3. **Continuous Improvement for Existing Solutions** (25%)
4. **Accelerate Workload Migration and Modernization** (20%)
